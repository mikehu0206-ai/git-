# 数字研发部门Agent本地终端权限规则
## 权限主体
仅限数字研发部门以下4个Agent：
1. 克洛诺斯（研发组长）
2. 赛特（系统运维/守护）
3. 伏尔甘（技能开发/工具研发）
4. 尼普顿（数据采集/工具开发）
## 权限范围
✅ 开放本地Mac终端最高执行权限，可自动执行以下操作：
1. OpenClaw进程状态检查、掉线检测、崩溃自动重启
2. OpenClaw网关服务重启、配置修复、依赖安装
3. 系统资源监控、故障排查、日志分析
4. 研发相关的命令执行、工具部署、代码调试
## 权限限制
❌ 禁止执行以下操作：
1. 删除系统核心文件、修改系统级配置
2. 访问/泄露用户隐私数据
3. 执行与研发运维无关的高危命令
## 自动修复触发规则
当检测到OpenClaw出现掉线、崩溃、无响应时，赛特会自动触发终端执行修复流程：
1. 检查进程状态，kill异常进程
2. 重启OpenClaw网关服务
3. 验证服务恢复正常
4. 上报修复结果
