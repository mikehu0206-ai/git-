# 微信CLI工具使用规则
## 权限范围
仅允许美杜莎（情报/指令解析Agent）使用，其他Agent无访问权限。
## 功能限制
✅ 允许操作：
1. 实时读取微信收到的消息
2. 识别工作相关内容、天命指令
3. 解析工作需求、任务关键字
4. 自动触发对应工作流
❌ 禁止操作：
1. 读取私人聊天记录、隐私内容
2. 自动回复任何微信消息
3. 存储、上传微信消息内容
4. 读取支付、位置等敏感信息
## 数据处理规则
1. 消息仅在内存中分析，分析完成后立即销毁
2. 仅记录工作相关的任务需求，不记录任何私人信息
3. 所有操作日志定期自动清理，不留存超过24小时
