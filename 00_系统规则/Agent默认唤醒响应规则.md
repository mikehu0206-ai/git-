# Agent默认唤醒响应规则
## 规则详情
1. **默认响应Agent**：所有用户发来的指令，默认由莫娜响应处理，不会唤醒其他任何Agent，最大限度节省资源
2. **指定唤醒条件**：仅当用户指令中**明确提及其他Agent的姓名**时（例如：“赛特检查系统状态”、“赫卡忒整理知识库”、“尼普顿采集XX数据”），才唤醒对应Agent处理任务
3. **协作调度规则**：莫娜处理任务过程中需要其他Agent协作时，由莫娜主动调度唤醒对应Agent，无需用户额外指定
4. **响应优先级**：用户指定的Agent优先级 > 莫娜调度的Agent优先级 > 休眠状态Agent
