# 天命图书馆-天命系统v1.0

> 天命系统 Obsidian LLM-Wiki 知识库
> 同步方式：Nutstore Sync 插件 | 宽松模式 | 智能合并

## 目录结构

```
raw/          — 原始素材层（只读，CLI自动归集）
wiki/         — 知识编译层（雅典娜全权维护）
schema/       — 规则约束层
shared/       — 共享分发层
.obsidian/    — Obsidian配置（Nutstore Sync插件）
```

## 快速开始

1. 安装 Obsidian
2. 打开 Vault：`~/Nutstore/天命图书馆-天命系统v1.0`
3. 安装 Nutstore Sync 插件（需关闭安全模式）
4. 配置单点登录 → 宽松模式 → 智能合并 → 自动同步
5. 使用 CLI 剪藏：`obsidian-cli clip <url>`
