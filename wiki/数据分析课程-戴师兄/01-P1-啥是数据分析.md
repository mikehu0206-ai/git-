# P1. 什么是数据分析

> 原视频：全网最良心的【数据分析自学课程】它来啦！P1. 啥是数据分析

## 核心定义：数据分析 = 观测 + 实验 + 应用

数据分析只不过是一种**客观认知世界并改造世界的基础方法**，实用高效非常简单。

## 案例：从小早餐店到连锁品牌上市

### 1. 初始阶段：隐性数据分析（没有电脑也会）

- 小店老板根据经验平衡产量：鲜肉包每天现包不做多，馒头冻存卖多少拿多少 → 避免浪费
- 节假日根据经验预估产量，多准备对应食材 → 这就是"预测分析"
- **结论：分析这件事你天生就会，就算没有电脑，大脑一样处理数据完成分析**

### 2. 第一个问题：房租上涨怎么办？

- 分类讨论（数据分析最基础方法）：列举所有解决方案，推演结果选收益最大方案
- 选项：换地址 / 提升营业额 → 换店风险太大，选提升营业额
- 外卖小哥建议做外卖 → 这就是 AB 测试（数据实验），哪个版本好就用哪个版本，大厂也这么干

### 3. 增长阶段：用数据发现问题解决问题

- 花了一千多买曝光，没人下单 → 通过公式拆解发现问题：
  ```
  营业额 = 曝光量 × 点击率 × 下单率 × 客单价
  ```
- 问题：只看曝光量（面子工程），没优化点击率，店铺封面/产品图片没优化+新店没评价 → 不敢下单
- 解决：请教同行，优化图片视觉，做评论，针对不同人群推不同套餐 → 营业额上来

### 4. 效率提升：自动化解放双手

- 多平台数据统计需要人工复制粘贴到 Excel，每天到凌晨
- 解决：RPA + Python 自动化脚本，自动汇总数据
- 进一步：BI 平台，实时刷新，多维度下钻分析
- **心得：当你每天重复处理大量数据，学会写脚本自动化，投入一次永久解放双手**

### 5. 开分店：选址问题

- 不能只数人流，要分析人流背后的转化逻辑 → 凌晨火车站是人流高峰但旅客不停留，早餐店开在大学城才对
- 数据帮助准确定位，开出成功分店

### 6. 连锁化：标准化管理

- 统一出品标准 → 自建数据库 → BI 看板实时监控
- 通过数据快速定位异常：
  - 看最大最小值定位异常门店
  - 看同比环比发现运营变动
  - 结合漏斗模型分析用户全流程转化

### 7. 最终：冲击上市

- 需要用数据展示过往业绩，用新技术讲未来故事 → 数据分析贯穿全程

## 数据分析在企业中的角色演变

### 数据循环三阶段

1. **数据建设**：打基础，搭建数据库/BI平台，保证数据被很好收集管理使用
2. **数据分析**：从数据提炼信息，发现规律验证假设，为决策提供依据
3. **业务执行**：根据洞察把策略落实到行动

### 企业发展过程中的分工演化

- 小店老板：自己一个人干全流程
- 店铺变大：分析和执行分离，有人专门研究数据
- 连锁品牌：需要数据工程师建设数据系统，演化出现代企业完整团队

### 常见分支角色

| 角色 | 职责 |
|------|------|
| 数据科学家 | 设计数据实验，验证策略有效性 |
| 数据分析师 | 定期做报表，监控业务，为业务保驾护航 |
| 数据运营 | 业务人员也要懂数据，边看数据边优化业务 |
| 算法工程师 | 海量数据用模型自动分析推荐，如个性化推荐 |
| 数据产品经理 / 数据工程师 | 搭建内部数据系统和工具（BI平台等），让业务方便看数据 |

## 数据分析是中华文化自古就有的智慧

- 三千年前：商人用结绳计数 → 现在的数据库
- 姜子牙：草船借箭前做天气预测 → 古代版数据建模
- 《齐民要术》：记载农时规律 → 古代统计分析报告

**核心：工具变了，本质没变——通过信息优势创造价值**

## 宏观视角：数据改变现代社会运行

- 国家层面：数据调配资源，高德地图实时交通
- 战争领域：信息化战场，数据先行，无人机精准打击
- 考古艺术：AI修复古画，文物成分分析 → 数据传承文明
- 工业金融：智能工厂零库存，金融每秒千万交易 → 我们正处在数据文明起点
